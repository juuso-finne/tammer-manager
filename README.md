# Tammer-Manager

## 1. Overview

Tammer-Manager is an Android-based chess tournament manager for club-level use. Its main functionalities are keeping score and generating the pairings for the next round. It supports importing a player list from an xlsx file, splitting players to groups, and exporting the results in a spreadsheet.

## 2. Build and run

Install [Android Studio](https://developer.android.com/studio) and follow the [instructions](https://developer.android.com/studio/run) to run the app in a virtual or physical device.
